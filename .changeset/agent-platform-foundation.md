Add the agent platform foundation layer across durable agent runs, advanced tool registries, safety policies, replay/evaluation, trace observability, provider parity, and fixture/report workflows.

This release adds compatible durable run primitives with idempotency, cooperative cancellation, and schema-versioned state; experimental advanced tool registry helpers with HTTP tools, permissions, audit metadata, registry inspection, and tool fixtures; stable safety policy helpers for approvals, redaction, and budget guards; stable replay, mock, evaluation, report, trace, cost, and latency helpers; and stable provider parity matrix rendering and drift reports.
