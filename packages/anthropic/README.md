# @zhivex-ai/anthropic

Anthropic adapter for Zhivex AI SDK.

The adapter includes first-class request handling for current Claude API families such as Claude Fable 5,
Claude Sonnet 5, Claude Opus 4.8, and Claude Haiku 4.5. Claude Mythos 5 can be addressed by model ID for
approved customers, but Anthropic treats Mythos access as limited rather than self-serve.

## Install

```bash
bun add @zhivex-ai/anthropic
```

Repository and full documentation:
