# @zhivex-ai/sdk

Recommended entry point for Zhivex AI SDK. Re-exports the high-level public API for text generation, streaming, structured output, tools, multimodal messages, and embeddings.

## Install

```bash
bun add @zhivex-ai/sdk
```

For the current release candidate:

```bash
bun add @zhivex-ai/sdk@next
```
