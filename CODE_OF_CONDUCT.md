# Code of Conduct

## Our Pledge

We want Zhivex AI SDK to be a welcoming, respectful, and constructive project for everyone.

Contributors, maintainers, and participants are expected to:

- be respectful and considerate,
- assume good intent,
- give feedback constructively,
- focus on the work rather than the person,
- help keep discussions productive and inclusive.

## Unacceptable Behavior

Examples of unacceptable behavior include:

- harassment, intimidation, or discrimination,
- hateful or demeaning language,
- personal attacks or trolling,
- publishing private information without permission,
- repeated disruptive or bad-faith behavior.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues, and other contributions that violate this Code of Conduct.

## Reporting

If you experience or witness behavior that violates this Code of Conduct, report it privately to the maintainers through the contact path described in `SECURITY.md` or the repository contact details.

All reports will be reviewed and handled as promptly and fairly as possible.
